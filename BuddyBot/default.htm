BuddyBot

An intelligent personal assistant bot to assist with your everyday needs

Meet Buddy

BuddyBot is a chatbot built with .NET and the Microsoft Bot Framework; he's a text-based intelligent personal assistant, who can understand what you write using a natural language processing service called LUIS. BuddyBot has many features such as retrieving weather forecasts, telling jokes, generating random values and is configurable through swappable personalities and a database managed small talk response service.
